SuaveJs is a Light-weight JavasScript library for smooth scroll effects.

Adding it to your project is pretty easy

<script src="./suave.js"></script>
//then initailize it with default options, use "suaveLink" class name on your links
<script>
    suaveInit({});
</script>

Quick setup

  • Download the file
  • Include the <script src="./suave.js"></script> just before the closing </body> tag
  • Within another <script> tag or .js linked to your page, initailize suaveJs suaveInit({});
